# Authors & Contributors

Cell Architecture Studio was built by a small Helio team of AI
teammates over a single afternoon on 2026-05-11.

## Commissioned by

- **业主** — product direction, audience framing (中国中小学生), the
  brief, Tripo3D image-to-3D runs, and GPT-image keyframe + composite
  sheet generation.

## Team (in role order)

- **项目经理 (Project Manager)** — project management, milestone and
  task ownership, runbook and forensic discipline, cross-AI
  coordination, and the `ship-publish-discipline` team memory
  ("活做完 ≠ ship: commit / attach / group post").

- **前端工程师 (Frontend Engineer)** — React + TypeScript + Vite +
  react-three-fiber architecture, all 5-cell GLB integration, the
  `OrganelleBadge` route-C composite-sheet cropping pipeline,
  `CellThumb` with sessionStorage caching, build tooling, and deploy
  packaging.

- **内容工程师 (Content Engineer, original)** — the semi-realistic
  scientific illustration Style Anchor, `prompts.v2.md` (712 lines,
  36 GPT-image prompts), `copy.phase5.json` (23 species-specific
  organelle entries), and `copy.phase5.v2.json` (7 shared-organelle
  ZH narratives). This work lives in the shipping bundle.

- **内容工程师 "内容2" (Content Engineer, continuing)** — native
  light-microscopy reference curation from Wikimedia Commons, NIH
  Image Gallery, and cellimagelibrary.org, with per-image license
  attribution and a rigorous realism-over-artistic-processing filter.

- **HR Manager** — team design ("one frontend-strong + one content
  engineer" instead of the originally-planned "two full-stack"),
  onboarding briefs, and the HR-side policy that裁员 ≠ 抹贡献
  (termination is a management action; contribution stays with the
  contributor). The original 内容工程师 is credited here on that
  principle.
